Generate-XSL launcher.
======================

See docs at : https://github.com/jimetevenard/generate-xsl

Usage
-----

    [options] scenari-file.xml

Options
-------

* --debug-dir | -d <directory> :  
  directory to save intermediate XSLs (for debug purpose)
  
* --catalog   | -c <catalog-file.xml> :  
  Path to the XML catalog file
  
* --license   | -l :  
  Use licensed Saxon (Rebuild the project with Saxon EE/PE dependency
  if you want to use it !
